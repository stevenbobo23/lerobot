API接口说明

GET /status - 获取机器人状态

POST /control - 控制机器人移动和机械臂

移动控制示例:
{
    "command": "forward",  // 可选值: forward, backward, left, right, rotate_left, rotate_right, stop
    "duration": 2.5       // 可选: 移动持续时间(秒)，默认为0(持续移动)
}

自定义速度控制:
{
    "x_vel": 0.2,     // 前后速度 (m/s)
    "y_vel": 0.0,     // 左右速度 (m/s) 
    "theta_vel": 0.0, // 旋转速度 (deg/s)
    "duration": 1.0   // 可选: 移动持续时间(秒)
}

机械臂位置控制:
{
    "arm_shoulder_pan.pos": 45,    // 肩膏水平旋转 (-100到100)
    "arm_shoulder_lift.pos": -30,  // 肩膏垂直旋转 (-100到100)
    "arm_elbow_flex.pos": 60,      // 肘关节 (-100到100)
    "arm_wrist_flex.pos": 20,      // 腕关节弯曲 (-100到100)
    "arm_wrist_roll.pos": 0,       // 腕关节旋转 (-100到100)
    "arm_gripper.pos": 50          // 夹爪 (0到100)
}
